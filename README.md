## Installation:

Before you begin, make sure you have Node Version Manager (nvm) installed on your system.

Install `nvm`
```
 git clone https://github.com/Paysharperly1/paysharperly-vendor-mobile
 ```
 ```
 nvm use 18
```
```
 npm ci
 ```
